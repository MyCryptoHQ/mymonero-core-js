# @xmr-core/xmr-varint

Create varints from numbers and strings for usage in Monero's data structures

## Usage

---

```ts
import { encode_varint } from "@xmr-core/xmr-varint";
const varint = encode_varint(10);
```

## Installation

---

```sh
yarn add @xmr-core/xmr-varint
```
